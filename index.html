The Maya developed glyphs that visually express the numeric values they represent.

  • Dots indicate values of one.
  • Lines indicate values of five.
  • A glyph with one dot and one line represents the value "six."

The Maya employed a vertically structured "base 20" value system.

  • This row represents the 400s place.
  • The next row represents the 20s place.
  • The bottom row represents the units, or 1s, place.
  • The stacked rows are added to obtain a sum.
